DG SURF (v1.0.0.3)

 

DG Surf is a new web browser provided by DG DEV. It currently does not support tabs or any other basic web browsers features. For now you can change search engines and home page. and enter URL for a website you want to visit. There is a separate search box on top to do general search with your selected search engine. It is created in Visual Studio 2017 IDE. It is written in Visual Basic .NET language. It is powered by our new engine dgs engine (dg surf engine). It is still in beta phase More features will come in the coming months. Currently it is availble for Windows operating system only.

Download DG Surf v1.0.0.3 (Windows)
